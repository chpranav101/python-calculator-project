# python calculator project using basic skills

**Description**
This project is a menu-driven smart calculator developed using Python.  It performs multiple mathematical operations like addition, subtraction,  multiplication, division, power, and modulus.  The calculator accepts multiple inputs from the user and uses NumPy  for efficient numerical operations.

**Features**
- Menu-driven program
- Supports multiple numbers
- Uses NumPy
- Handles division by zero

**Technologies Used**
- Python
- NumPy

**How to Run**
1. Install Python
2. Install NumPy: pip install numpy
3. Run the file:
   python calculator.py

**Minimum Required**
This project is a Python-based calculator using NumPy.
